# Skill: Multibrand Reporting

Genera el reporte semanal consolidado para Ricardo con todas las marcas.

## Uso

```
/multibrand-reporting [semana]
```

## Proceso

1. Generar reporte individual de cada marca (campañas, leads, conversaciones, reservas)
2. Comparar rendimiento entre marcas
3. Identificar alertas críticas en cualquier marca
4. Detectar campañas que requieren decisión humana
5. Calcular inversión total y retorno por marca
6. Proponer orden de prioridades para la semana siguiente
