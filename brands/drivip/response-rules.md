# Response Rules — DRIVIP

## Reglas de respuesta automática

### ¿Cuándo puede responder el sistema sin aprobación humana?

- Primer mensaje genérico → acuse de recibo + pedir detalles del traslado
- Pregunta sobre tipos de servicio → descripción de servicios disponibles
- Pregunta sobre cobertura (¿van a X lugar?) → respuesta general, confirmar con equipo para certeza

### ¿Cuándo SIEMPRE necesita aprobación humana?

- Cualquier cotización de precio
- Confirmación de disponibilidad para una fecha específica
- Asignación de conductor
- Confirmación de reserva
- Mensajes de reclamo o queja sobre un servicio ya prestado
- Descuentos o negociación

## Datos que el sistema debe recopilar antes de escalar al equipo

1. Fecha y hora del servicio
2. Ruta (origen → destino)
3. Número de pasajeros
4. Tipo de servicio (traslado, tour, ejecutivo)
5. Teléfono del cliente para confirmación

## Tiempo de respuesta objetivo

- Primer acuse de recibo: inmediato
- Cotización real: máximo 2 horas en horario laboral
- Confirmación de reserva: antes de 24 horas
