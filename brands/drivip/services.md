# Services — DRIVIP

## Servicios principales

- **Traslados aeropuerto**: desde y hacia el Aeropuerto Internacional de Tocumen y aeropuertos regionales
- **Traslados punto a punto**: cualquier ruta dentro de Panamá
- **Tours privados**: tours personalizados dentro de Panamá
- **Servicio ejecutivo**: transporte para reuniones, eventos corporativos
- **Transportación para eventos**: bodas, graduaciones, cumpleaños

## Reglas comerciales

- Los precios se dan después de conocer: ruta, fecha, hora y número de pasajeros
- La disponibilidad la confirma el equipo operativo, no el sistema
- Los conductores los asigna el equipo, no el sistema
- Las reservas se confirman con pago anticipado (monto lo define el equipo)

## Proceso de venta típico

1. Cliente contacta por WhatsApp o Instagram
2. Sistema pide información: fecha, hora, ruta, pasajeros
3. Equipo verifica disponibilidad y cotiza
4. Cliente confirma y paga
5. Se asigna conductor y se envía confirmación
